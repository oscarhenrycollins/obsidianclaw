What's OpenClaw?

OpenClaw is a personal AI agent that runs on your own machine. It remembers everything, handles tasks on its own, reads and writes files, searches the web, and keeps working while you sleep. You can talk to it over Telegram, Signal, or a web app.

This plugin adds one more way to talk to it: directly inside Obsidian.

Telegram / Signal

A single chat thread. You send a message, you get a reply. You can't see what the agent is reading or writing. Your notes are somewhere else.

This plugin

Your notes are on the left, your agent is on the right. Send any note as context with one command. See every file it reads, every tool it runs. Everything in one place.

Setup

1

Install the plugin

Add it through BRAT in Obsidian. One line to paste.

2

Paste your gateway URL and token

Same credentials you use for any OpenClaw client. Connection is encrypted via Tailscale.

3

Approve the device

Tell your agent "approve my device" or run one CLI command. Done once per device.

No servers in between. Your messages go directly from Obsidian to your OpenClaw instance over an encrypted Tailscale connection. Nothing touches our infrastructure. The plugin is free and open source.
